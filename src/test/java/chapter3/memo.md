##### 구현하기 쉬운 테스트부터 시작하기

- 모든 조건을 충족하는 경우
- 모든 조건을 충족하지 않는 경우

##### 테스트 작성 순서 연습
- 서비스를 사용하려면 매달 1만 원을 선불로 납부한다. 납부일 기준으로 한 달 뒤가 서비스 만료일이다.
- 2개월 이상 요금을 납부할 수 있다.
- 10만 월을 납부하면 서비스를 1년 제공한다.
